Bug Tracker

Bug Tracker is a full-stack development project. The purpose of this project was to allow software developers to manage bugs and other issues through the use of tickets.

The full website is available here.

* Registration is not currently open to the public. A demo is available upon request.

Technology

  • Cloudfire Store
  • Angular
  • Firebase
  • Some of the JavaScript Libraries used in this project include: RxJS, AngularFire, Charts.JS, etc.

Design Process

The process for this project was pretty standard, I sketched a wireframe for both mobile and desktop versions. I always design with mobile in mind so, the only difference for mobile is the layout of the dashboard. The final website looks very similar to this wireframe.
