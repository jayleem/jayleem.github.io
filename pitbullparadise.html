Pitbull Paradise

Pitbull Paradise was a full-stack development project. The purpose of this project was to apply my knowledge of both front-end design as well as back-end development. The goal of the website was to allow users to be able to find relevant information about the resuce such as mission statement and goals of the organization etc., but perhaps more importantly, for users to browse dogs that are available for adoption and to then be able to apply for adoption online.

The full website is available here.

Technology

  • MongoDB Cloud
  • Express 4x
  • Angular
  • Node.js

Authentication

The adminstration dashboard was created to allow remote management over the website and provide useful real-time data visualization using Charts.js library. I wanted to restrict this section of the website to authorized-users only. I accomplished this by authenticating login credentials via on the server-side using the JWT library.
